# Google Sheets 데이터 연결 테스트 가이드

## ✅ CSV 파일 업로드 완료!

이제 Google Sheets에 담보별 보험료 데이터가 포함되어 있습니다.

---

## 🧪 테스트 방법

### 방법 1: API 테스트 엔드포인트

브라우저에서 열기:
```
http://localhost:3000/api/test-apis
```

또는 터미널에서:
```bash
curl http://localhost:3000/api/test-apis
```

**확인 사항:**
- ✅ Gemini API 연결
- ✅ Google Sheets API 연결
- ✅ Google Custom Search API 연결
- ✅ **담보별_보험료 데이터 개수** 확인

### 방법 2: 실제 블로그 생성 테스트

1. 대시보드에서 블로그 생성
2. 주제 입력: 예) "30세 남성 암보험 비교"
3. AI 글쓰기 시작
4. 생성된 블로그에서 확인:
   - 담보별 보험료 비교 테이블이 포함되었는지
   - 각 담보별 상세 정보가 있는지

---

## 📊 확인 사항

### Google Sheets 데이터 확인
- `담보별_보험료` 시트에 1960개 데이터가 있는지
- 컬럼 구조가 올바른지:
  - 연령 | 성별 | 담보명 | 가입금액 | 보험사 | 보험료

### 블로그 생성 시 확인
- 담보별 보험료 비교가 포함되는지
- 정확한 연령/성별 데이터가 사용되는지

---

## 🔍 예상 결과

블로그 생성 시:
- **30세 남성** 주제 → 30세 남성 데이터 자동 사용
- **40세 여성** 주제 → 40세 여성 데이터 자동 사용
- 담보별 보험료 상세 비교 포함
- 각 보험사별 담보 보험료 표시

---

## ⚠️ 문제 발생 시

1. **데이터가 안 보임**: Google Sheets에서 시트 이름 확인 (`담보별_보험료`)
2. **잘못된 데이터**: CSV 파일 재업로드
3. **연결 오류**: API Key 확인 (`.env.local`)
