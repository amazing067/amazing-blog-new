## 📊 실제 카페 답변 분석

### 실제 보험카페 답변 길이 (샘플 분석)

**간단한 질문 답변 (50-80자):**
```
"미라클님~ 해당 보험사는 3백만원까지 가입 가능합니다 
누적 확인이 필요해서 오픈카톡에 정보 남겨주시면 연락 드리겠습니다^^"
```
→ 약 60자, 핵심 정보만

**일반적인 질문 답변 (80-120자):**
```
"생일선물님~ 10대골절진단비는 일반 골절시 50만원, 
10대 골절에 해당되는 경우 250만원 보장됩니다^^ 
치아파절 포함된 일반 골절도 포함되어 있어요"
```
→ 약 90자, 핵심 + 기본 정보

**복잡한 질문 답변 (120-150자):**
```
"크리스마스카운터님~ 암주요치료비 특약은 암수술, 항암약물치료, 
항암방사선치료만 포함됩니다 중입자치료는 별도 특약으로 
추가 가입이 가능한 경우가 있어요 보험료도 월 3만원대로 
합리적인 편입니다"
```
→ 약 130자, 핵심 + 기본 + 상세 정보

**결론:**
- ✅ 대부분의 답변이 50-150자 범위 내
- ✅ 핵심, 기본정보, 상세정보 모두 50-150자에 포함 가능
- ✅ 200자 이상 답변은 거의 없음

## 🔀 옵션 1: 통일 (50-150자, 최대 200자)

### 구조
```
단일 답변 길이: 50-150자 (최대 200자 절대 초과 금지)
- 핵심 정보 필수 포함
- 기본 정보 포함
- 상세 정보는 질문 복잡도에 따라 자동 조절
```

### 장점 ✅
1. **단순함**: 사용자가 선택할 필요 없음
2. **실제 카페와 유사**: 대부분의 답변이 이 범위
3. **일관성**: 모든 답변이 비슷한 길이
4. **자동 조절**: 질문이 간단하면 짧게, 복잡하면 길게 (범위 내)
5. **사용자 혼란 없음**: 옵션 선택 불필요

### 단점 ❌
1. **복잡한 질문도 짧게**: 매우 복잡한 질문도 150자 제한
2. **유연성 부족**: 사용자가 원하는 길이 선택 불가

### 구현 방법
```typescript
// 프롬프트
7. **답변 길이 (매우 중요! 실제 카페 답변 기준)**:
   - **⚠️ 반드시 50-150자 내외로 작성하세요 (최대 200자 절대 초과 금지!)**
   - 질문이 간단하면 50-80자 (핵심만)
   - 질문이 일반적이면 80-120자 (핵심 + 기본 정보)
   - 질문이 복잡하면 120-150자 (핵심 + 기본 + 상세 정보)
   - 핵심 정보는 반드시 포함 (구체적 금액, 조건, 특약명)
   - 기본 정보는 가능한 한 포함 (비교, 추가 설명)
   - 상세 정보는 질문 복잡도에 따라 포함 (이유, 맥락)

// API 후처리
const MAX_LENGTH = 200
if (answerContent.length > MAX_LENGTH) {
  // 문장 단위로 자르기
  answerContent = truncateBySentence(answerContent, MAX_LENGTH)
}
```

## 🔀 옵션 2: 두 개로 나누기 (짧은/기본)

### 구조
```
짧은 답변: 50-100자 (핵심만)
기본 답변: 100-150자 (핵심 + 기본 + 상세)
```

### 장점 ✅
1. **선택권 제공**: 사용자가 원하는 길이 선택 가능
2. **명확한 구분**: 짧은/기본 명확히 구분
3. **유연성**: 상황에 따라 선택 가능

### 단점 ❌
1. **사용자 혼란**: 어떤 걸 선택해야 할지 모름
2. **실제 카페와 다름**: 실제로는 선택 옵션이 없음
3. **복잡성 증가**: UI에 옵션 추가 필요
4. **일관성 부족**: 답변 길이가 들쭉날쭉
5. **의사결정 부담**: 매번 선택해야 함

### 구현 방법
```typescript
// 프롬프트
7. **답변 길이**: ${answerLength === 'short' ? `
   - **⚠️ 반드시 50-100자 내외로 작성하세요 (핵심만)**
   - 핵심 정보만 간결하게 (구체적 금액, 조건, 특약명 필수)
   - 기본 정보는 최소화
   - 상세 정보는 제외
   ` : `
   - **⚠️ 반드시 100-150자 내외로 작성하세요 (핵심 + 기본 + 상세)**
   - 핵심 정보 필수 포함 (구체적 금액, 조건, 특약명)
   - 기본 정보 포함 (비교, 추가 설명)
   - 상세 정보 포함 (이유, 맥락)
   `}
```

   - UI 옵션 제거 가능
   - 유지보수 용이

4. **품질 보장**
   - 50-150자 범위 내에서 핵심, 기본, 상세 모두 포함 가능
